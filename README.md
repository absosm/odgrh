# Node Package Manager (npm):
```
npm install -g firebase-tools
```
## Firebase Token
```
firebase login:ci
```
## Authentification
```
firebae login
```
## Dependencies
```
npm install
```

## Testing in local emulator

### configure emulator:
```
firebase init emulators
```
### start emulator:
```
firebase emulators:start
```

## Deploy
```
firebase deploy
```
